# Flare — Server Management AI Agent

A lightweight VPS management agent with a Bubbletea TUI.

## Quick Start

```bash
flare setup     # First-time configuration
flare chat      # Interactive chat
flare daemon    # Background monitoring
```

## Features

- Interactive chat with LLM agent
- Run shell commands, read/write files, manage services
- Background monitoring with health checks
- Webhook alerts for critical events
- Server context memory for informed responses
